My Projects

Project 1:Temperature convertor

The "Temperature Converter" project is a web-based utility developed using HTML, CSS, and JavaScript. It allows users to easily convert temperatures between Celsius, Fahrenheit, and Kelvin. Users input a value in one unit, click "Convert," and instantly see the equivalent temperature in the desired unit. This interactive tool is both educational and practical, making temperature conversions quick and straightforward.

Project 2:To do list

The "To-Do List" project is a web application created with HTML, CSS, and JavaScript. It provides users with a user-friendly interface to manage tasks and stay organized. Users can add, edit, and mark tasks as complete, facilitating efficient task management and productivity.
